## Why

Hone needs explicit boundaries for fitness, diet, and sport medic agent behavior to prevent unsafe overlap and ensure predictable escalation paths.

## What Changes

- Define allowed and prohibited behavior per domain agent.
- Define cross-agent handoff triggers and escalation rules.
- Define mandatory safety language for high-risk contexts.
- Define constraints that prevent medical overreach from non-medical agents.

## Capabilities

### New Capabilities
- `domain-agent-capabilities`: Domain-specific behavior contracts and escalation rules for fitness, diet, and sport medic agents.

### Modified Capabilities
- None.

## Impact

- Affects agent policy, routing logic, and response safety behavior.
- Aligns domain outputs with verification and strict-gate policy.
